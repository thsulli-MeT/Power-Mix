sMV Power Mix v0.28

Uses audio/library.json manifest for GitHub Pages-friendly library + preloads.
Preload A: dj_Samples_1.mp3
Preload B: dj_Samples_2.mp3
Sample pads preload: whistle_1, chirp, Scratch_3, phaser, scratch_fast, itchy, drama, pullover

Controls:
- Shift+Click sample pad: load local sample
- Option/Alt+Click sample pad: choose from audio/library.json samples
- Press L: open track library

Run:
  python3 -m http.server 8080
